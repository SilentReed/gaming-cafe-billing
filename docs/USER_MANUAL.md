## 超管操作指南

### 平台概览

登录后默认显示平台概览页面，包含：
- 商户总数统计
- 营业中商户数
- 用户总数
- 商户列表快速查看

### 商户管理

#### 创建商户

1. 点击「商户管理」菜单
2. 点击「新增商户」按钮
3. 填写商户信息：
   - 商户名称（必填）
   - 联系人
   - 联系电话
   - 地址
4. 设置登录账户：
   - 用户名（必填）
   - 密码（必填）
5. 点击「创建」

#### 编辑商户

在商户卡片上点击「编辑」按钮，修改商户基本信息。

#### 设置授权期限

1. 在商户卡片上点击「设置期限」
2. 选择到期日期
3. 点击「保存」
4. 留空表示永久授权

> ⚠️ 授权到期后，商户所有功能将无法使用，登录后会提示续费。

#### 管理商户账户

1. 点击商户卡片上的「账户」按钮
2. 可以：
   - 查看所有账户
   - 新增账户
   - 编辑账户信息
   - 重置密码
   - 删除账户（仅限员工）

### 功能配置

控制每个商户可用的功能模块：

1. 点击「功能配置」菜单
2. 找到目标商户
3. 点击「配置功能」
4. 勾选/取消勾选功能模块
5. 点击「保存配置」

**可配置功能：**
- 收银台/计费大厅
- 会员管理
- 账单记录
- 时段套餐
- 餐饮商品
- 订单管理
- 交班管理
- 员工管理
- 报表统计
- 预约管理
- 主机设置

## 商户管理员操作指南

### 收银台

收银台是日常操作的主界面，显示所有主机状态：

#### 主机状态说明

| 状态 | 颜色 | 说明 |
|------|------|------|
| 空闲 | 绿色 | 可以上机 |
| 使用中 | 蓝色 | 有顾客在使用 |
| 暂停中 | 黄色 | 计时已暂停 |
| 维护 | 灰色 | 主机维护中 |

#### 开台操作

1. 点击空闲的主机卡片
2. 选择会员（可选，散客直接跳过）
3. 选择计费方式：
   - 正计时：按实际使用时间计费
   - 倒计时：预付固定时长
4. 选择套餐（可选）
5. 点击「开始计费」

#### 结账操作

1. 点击使用中的主机卡片
2. 查看消费详情
3. 选择支付方式
4. 点击「结账」

#### 暂停/恢复

- 点击主机卡片 → 「暂停」
- 暂停后点击 → 「恢复」

### 计费大厅

实时查看所有主机的计费状态：
- 当前使用时长
- 预计费用
- 会员信息

### 会员管理

#### 添加会员

1. 点击「会员管理」菜单
2. 点击「新增会员」
3. 填写会员信息：
   - 姓名
   - 手机号
   - 备注
4. 点击「保存」

#### 会员充值

1. 在会员列表找到目标会员
2. 点击「充值」
3. 输入充值金额
4. 选择支付方式
5. 确认充值

#### 会员等级

系统自动根据累计消费升级会员等级：

| 等级 | 折扣 | 门槛 |
|------|------|------|
| 普通会员 | 10折 | ¥0 |
| 银卡会员 | 9.5折 | ¥500 |
| 金卡会员 | 9折 | ¥2000 |
| 钻石会员 | 8.5折 | ¥5000 |

### 账单记录

查看所有消费记录：
- 按日期筛选
- 按状态筛选（已结/未结）
- 导出账单

### 时段套餐

创建包时套餐：

1. 点击「时段套餐」菜单
2. 点击「新增套餐」
3. 填写：
   - 套餐名称
   - 时长（小时）
   - 价格
4. 保存

### 餐饮商品

管理餐饮商品：

1. 点击「餐饮商品」菜单
2. 点击「添加商品」
3. 填写：
   - 商品名称
   - 分类
   - 价格
   - 库存
4. 保存

### 报表统计

查看经营数据：
- 日/周/月营收
- 主机使用率
- 会员消费排行
- 热门时段分析

### 交班管理

1. 点击「交班」菜单
2. 查看当前班次统计
3. 点击「交班」
4. 确认交班信息

### 员工管理

#### 添加员工

1. 点击「员工管理」菜单
2. 点击「新增员工」
3. 填写：
   - 用户名（登录用）
   - 密码
   - 姓名
   - 角色（管理员/员工）
   - 手机
4. 点击「创建」

#### 编辑员工

点击员工列表中的「编辑」按钮，修改员工信息。

#### 重置密码

点击员工列表中的「重置密码」按钮，设置新密码。

#### 删除员工

点击员工列表中的「删除」按钮（仅限普通员工）。

## 员工操作指南

## 常见问题

### Q: 忘记密码怎么办？

**超管密码：**
需要直接修改数据库：
```bash
cd backend
python -c "
from app.utils.auth import hash_password
import sqlite3
conn = sqlite3.connect('gaming_cafe.db')
hashed = hash_password('new_password')
conn.execute('UPDATE users SET password_hash = ? WHERE username = ?', (hashed, 'admin'))
conn.commit()
"
```

**商户/员工密码：**
超管登录后，在商户管理中重置密码。

### Q: 商户授权过期怎么办？

1. 超管登录
2. 进入「商户管理」
3. 找到过期商户
4. 点击「设置期限」
5. 延长到期日期

### Q: 如何备份数据？

```bash
# 备份数据库
cp backend/gaming_cafe.db backup/gaming_cafe_$(date +%Y%m%d).db

# 恢复数据库
cp backup/gaming_cafe_20260630.db backend/gaming_cafe.db
```

### Q: 如何配置 HTTPS？

参考 README.md 中的 Nginx 反向代理配置，使用 Let's Encrypt 或其他证书。

### Q: 如何查看系统日志？

```bash
# Docker 部署
docker-compose logs -f

# 手动部署
# 日志输出到终端
```

### Q: 移动端如何使用？

系统支持响应式设计，手机浏览器直接访问即可。
- 点击右上角用户图标可退出登录
- 侧边栏可收缩为图标模式

### Q: 如何添加新的游戏主机？

在「主机设置」页面：
1. 点击「添加主机」
2. 填写主机名称
3. 选择主机类型
4. 设置费率
5. 保存

### Q: 支持哪些支付方式？

系统支持自定义支付方式，在「系统配置」中设置：
- 现金
- 微信
- 支付宝
- 会员余额
- 其他
